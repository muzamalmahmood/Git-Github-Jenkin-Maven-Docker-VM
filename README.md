# Git-Github-Jenkin-Maven-Docker-VM
Built a CI/CD pipeline using Git, GitHub, Jenkins, Maven, Docker, and Virtual Machine. The project automates code build, testing, and deployment by integrating version control, build tools, and containerization for efficient and reliable delivery.
